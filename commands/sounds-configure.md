# Configure Claude Code Sound Effects

Run the interactive sound configuration dialog to customize sounds and speech for different events.

Execute this command to open the configuration menu:

```bash
~/.claude/sounds/configure.sh
```

The configuration menu allows you to:
- Toggle sounds ON/OFF globally
- Toggle speech ON/OFF
- Adjust master volume (0.25 - 1.0)
- Configure individual event sounds
- Preview all available sounds
- Preview all available voices
- Reset to default settings

Current settings are stored in `~/.claude/sounds/config.json`
